==========================
Secondary Bootloader (SBL)
==========================

SBL выполняется на RISC0 из QSPI0 в режиме XIP (загрузка напрямую из QSPI0).

SBL выполняет:

* копирование кода инициализации DDR из QSPI0 в SRAM;
* запуск кода инициализации DDR из SRAM (инициализатор DDR должен вернуть управление SBL после завершения работы);
* копирование TF-A из QSPI0 в DDR;
* копирование U-Boot из QSPI0 в DDR;
* запуск TF-A на ARM CPU.

Данный проект не использует стандартную библиотеку C и поэтому может быть скомпилирован, используя тулчейн Linux.
